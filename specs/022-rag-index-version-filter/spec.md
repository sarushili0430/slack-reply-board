# 022 RAG Index Version Filter

## Functional Requirements

### FR-RAG-002 Retrieval uses the active index version

RAG retrieval must exclude chunks from inactive index versions when embedding or retrieval models
change.

### AC-RAG-002-01 Inactive index versions are excluded

Given:

- The vector store contains chunks from multiple index versions.
- The active index version is `embedding-v2`.

When:

- Retrieval candidates are prepared.

Then:

- Only chunks with index version `embedding-v2` remain.
- Chunks from older or unrelated index versions are excluded.

## Security Impact

- Retrieval does not mix embeddings generated by incompatible model versions.
- Draft grounding and ranking are based on the active index version only.
